# 🌱 Farm-Connect

Full Stack Agricultural Marketplace Platform built with React, Spring Boot, MySQL, JWT Authentication and Role-Based Access Control.

## ⭐ Key Highlights

* Full Stack Agricultural Marketplace Platform
* Secure JWT Authentication & Authorization
* Role-Based Access Control (Admin, Farmer & Buyer)
* RESTful API Integration
* Product Management System
* Order Management Workflow
* Responsive User Interface
* MySQL Database Integration

---

## 🚀 Features

### Buyer
- Register and Login
- Browse Products
- Search and Filter Products
- Place Orders
- View My Orders
- JWT Authentication

### Farmer
- Register and Login
- Add Products
- Edit Products
- Delete Products
- Manage Orders
- Approve/Reject Orders

### Admin
- Manage Users
- Manage Products
- Manage Orders
- Dashboard Analytics
